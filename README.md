# Empty Trash

**Empty Trash** is a minimalist menu bar utility designed for macOS. It automates the process of emptying the Trash, eliminating the need to do it manually by clearing it automatically after files are deleted.

## Features

- **Real-Time Auto Empty**  
  Whenever you move files to the Trash, the app automatically empties it, keeping the Trash always clean.

- **One-Click Launch at Login**  
  Easily control whether the app starts with the system directly from the menu bar, making it convenient for long-term background operation.

- **Smart Permission Guidance**  
  On first launch or when permissions are missing, the app automatically guides you to the system privacy settings to ensure proper functionality.

- **Minimalist Interaction Design**  
  No windows, no ads, no unnecessary configuration. All actions are handled through the menu bar icon, keeping your system interface clean.
